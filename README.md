# JCSVMatrixTransfer
一个方便将Renderdoc中导出的模型对其到世界空间位置的工具

（用于把多个Mesh对其到一起，方便逆向）

## 使用方式
- 拷贝 `JCSVMatrixTransfer` 文件夹到你的Unity项目中。
- 通过点击 `Jtools`-`J CSV Matrix Transfer`打开工具面板。
- 正确填入数据，转换。
